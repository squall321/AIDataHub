구현 기능 & 할 수 있는 것

전 기능 구현·실서버 동작 검증 완료. 외부(웹/Enterprise) 노출만 인프라 전제 대기.

데이터 적재 파이프라인

  • 입력: .docx .xlsx .pptx .md .pdf 5종
  • 출력: 단일 표준 JSON 스키마 (섹션·블록 단위)
  • 적재 즉시 e5-base 768d 임베딩 자동 인덱싱
  • 10자리 시퀀스 채번 · doc_type 분류 체계

검색 (3종 결합)

  • semantic — pgvector cosine 유사도
  • FTS — PostgreSQL tsvector 전문검색
  • tag — 정확 매칭 + 태그 부스트
  • score_threshold / max_depth 로 정밀도 제어

에이전트형 RAG (차별점)

  • recommend_agents(q) — 질문→적합 에이전트 자동 라우팅
  • retrieval_config: top_k · score_threshold · data_type_filter · tag_boost · max_depth
  • response_config: refuse_below_score (근거 부족 시 거절)
  • 출처(record id §section) 인용 강제 → 환각 억제

계층 · 샘플 학습

  • parent_record_id + depth (campaign 0 / specimen 1)
  • suggest-parent 자동 추천
  • sample_queries 임베딩으로 라우팅 정확도 보강
  • 에이전트 변경 이력(history) 추적
